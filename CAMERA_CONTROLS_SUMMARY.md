# Panasonic Camera Control Options Summary

## Choose options by number/letter to add to the new bottom panel menu

---

## 1. EXPOSURE CONTROL

**A. Iris (Aperture)**
- A1. Iris Mode (Auto/Manual)
- A2. Iris Value (F1.6 to F16)
- A3. Iris Speed (Fast/Normal/Slow)

**B. Gain**
- B1. Gain Mode (Auto/Manual)
- B2. Gain Value (-3dB to +42dB)
- B3. Gain Limit (max auto gain)

**C. Shutter Speed**
- C1. Shutter Mode (Auto/Manual)
- C2. Shutter Speed (1/100 to 1/10000)
- C3. Slow Shutter (1/30 to 1/1.25)

**D. ND Filter**
- D1. ND Filter (Off/1/4/1/16/1/64)
- D2. ND Filter Mode (Auto/Manual)

**E. Exposure Compensation**
- E1. Exposure Compensation (-2.0 to +2.0 EV)

---

## 2. COLOR & WHITE BALANCE

**F. White Balance**
- F1. WB Mode (Auto/Indoor/Outdoor/Manual/ATW)
- F2. Color Temperature (2000K-15000K)
- F3. Red Gain (-99 to +99)
- F4. Blue Gain (-99 to +99)
- F5. WB Speed (Fast/Normal/Slow)

**G. Color Matrix**
- G1. Matrix Type (Normal/EBU/NTSC/User)
- G2. Saturation (-99 to +99)
- G3. Hue (-99 to +99)

**H. Gamma**
- H1. Gamma Mode (Standard/Cinema/Wide/HD/FILMLIKE/etc.)
- H2. Gamma Level (Low/Mid/High)

**I. Black Balance**
- I1. Black Balance (Auto/Manual)
- I2. Master Black (-99 to +99)
- I3. Red/Blue Black (-99 to +99)

---

## 3. IMAGE ENHANCEMENT

**J. Detail (Sharpness)**
- J1. Detail Level (-99 to +99)
- J2. H/V Ratio (horizontal/vertical balance)
- J3. Crispening (detail threshold)
- J4. Coring (noise reduction threshold)

**K. Knee (Highlight Compression)**
- K1. Knee Mode (Off/Auto/Manual)
- K2. Knee Point (70% to 105%)
- K3. Knee Slope (adjustable curve)

**L. Noise Reduction**
- L1. DNR Level (Off/Low/High)
- L2. 2D/3D DNR (separate controls)
- L3. Temporal NR (frame-to-frame)

**M. Other Enhancements**
- M1. White Clip (100% to 109%)
- M2. Chroma Level (-99 to +99)
- M3. Color Gain (R/G/B individual)

---

## 4. PTZ CONTROL (Advanced - beyond basic joystick)

**N. Pan/Tilt Advanced**
- N1. Pan/Tilt Speed (1-24)
- N2. Pan/Tilt Position (absolute)
- N3. Pan/Tilt Limits (boundaries)

**O. Zoom Advanced**
- O1. Zoom Speed (1-7)
- O2. Zoom Position (absolute 0-16384)
- O3. Zoom Ratio Display
- O4. Digital Zoom (Enable/Disable)

**P. Focus Advanced**
- P1. Focus Mode (Auto/Manual)
- P2. Focus Speed (1-7)
- P3. Focus Position (absolute)
- P4. AF Sensitivity (High/Normal/Low)
- P5. AF Area (Wide/Center/Spot)
- P6. One Push AF (trigger)

---

## 5. PRESETS & SCENE FILES

**Q. PTZ Presets**
- Q1. Preset Save (1-100 presets)
- Q2. Preset Recall (1-100)
- Q3. Preset Name (assign names)
- Q4. Preset Thumbnail (capture/store)

**R. Scene Files**
- R1. Scene File Save (1-6 scene files)
- R2. Scene File Recall (1-6)
- R3. Scene File Name (assign names)

---

## 6. CAMERA OPERATIONS

**S. Recording (if supported)**
- S1. SD Recording (Start/Stop)
- S2. Recording Format (codec/format)
- S3. Recording Quality (resolution/bitrate)

**T. Streaming**
- T1. Stream Control (enable/disable)
- T2. Stream Resolution
- T3. Stream Bitrate
- T4. Stream Codec (H.264/H.265)

**U. Power & Status**
- U1. Power Control (ON/Standby)
- U2. Status Query (get camera status)
- U3. Firmware Version
- U4. Model Information

---

## 7. ADVANCED FEATURES

**V. Multi-Matrix**
- V1. Multi-Matrix Mode (enable/disable)
- V2. Multi-Matrix Settings (color temp adjustments)

**W. Shading**
- W1. Shading Mode (enable/disable)
- W2. Shading Adjustment (fine-tune)

**X. Image Transform**
- X1. Image Flip (Horizontal/Vertical)
- X2. Image Rotation (90°/180°/270°)

**Y. Privacy Masking**
- Y1. Privacy Mask (enable/disable)
- Y2. Mask Position (coordinates)
- Y3. Multiple Masks (zones)

**Z. Motion Detection**
- Z1. Motion Detection (enable/disable)
- Z2. Sensitivity
- Z3. Detection Area (zones)

---

## QUICK SELECTION OPTIONS

**Option 1: Essential Controls Only**
- A1-A2 (Iris Mode/Value)
- B1-B2 (Gain Mode/Value)
- C1-C2 (Shutter Mode/Speed)
- F1-F2 (WB Mode/Color Temp)
- P1-P6 (Focus Controls)
- Q1-Q2 (PTZ Presets)

**Option 2: Professional Controls**
- All of Option 1, plus:
- D1-D2 (ND Filter)
- F3-F4 (R/B Gain)
- H1 (Gamma Mode)
- J1 (Detail Level)
- K1-K2 (Knee Mode/Point)
- R1-R2 (Scene Files)

**Option 3: Full Feature Set**
- All controls from all categories

**Option 4: Custom Selection**
- Pick individual items by their letter/number codes
