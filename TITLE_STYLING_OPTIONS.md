# Setting Title Styling Options

Choose how to distinguish setting titles (like "Iris Mode", "Gain", "Shutter Mode", etc.) from regular text and control labels.

## Option 1: Bold Text (Current)
- **Style**: `font-weight: 600;`
- **Appearance**: Titles are bold but same color as regular text
- **Subtle distinction**

## Option 2: Orange Accent Color
- **Style**: `color: {COLORS['primary']};` (orange #FF9500)
- **Appearance**: Titles in orange, regular text in white
- **Clear visual distinction**

## Option 3: Larger Font Size
- **Style**: `font-size: 16px;` (instead of 14px)
- **Appearance**: Titles slightly larger than regular text
- **Size-based hierarchy**

## Option 4: Bold + Orange
- **Style**: `font-weight: 600; color: {COLORS['primary']};`
- **Appearance**: Bold orange titles
- **Strongest visual distinction**

## Option 5: Uppercase + Bold
- **Style**: `font-weight: 600; text-transform: uppercase;`
- **Appearance**: Bold uppercase titles (e.g., "IRIS MODE")
- **Typographic emphasis**

## Option 6: Underline
- **Style**: `text-decoration: underline;`
- **Appearance**: Underlined titles
- **Classic emphasis**

## Option 7: Background Box
- **Style**: `background-color: {COLORS['surface_light']}; padding: 4px 8px; border-radius: 4px;`
- **Appearance**: Titles in light grey boxes
- **Boxed emphasis**

## Option 8: Orange Background Box
- **Style**: `background-color: {COLORS['primary']}; color: {COLORS['background']}; padding: 4px 8px; border-radius: 4px;`
- **Appearance**: Titles in orange boxes with dark text
- **High contrast**

**Reply with the option number (1-8) to apply that styling to all setting titles.**
